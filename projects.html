Hex Chess Program Based on UCT Algorithm.
  • A collaborative project.
  • Won the Second Prize in the 16th Chinese Computer Games Championship with this project.
  • Implemented the Monte Carlo Search Tree algorithm and utilized the UCB formula to select the most promising child nodes for further exploration.
  • Developed an evaluation function in the paper, which is based on a breadth-first search (BFS) approach.
A Multi-cycle Processor using Verilog HDL
  • Implemented a processor with multi-cycle pipeline, including the Fetch stage, Decode stage, Execute stage, Memory Access stage, and Write Back stage.
  • Supported the MIPS-Lite4 instruction set.
  • Multi-cycle processors consisted of a datapath and a controller.The datapath included the following modules: PC (Program Counter), GPR (General Purpose Register), ALU (Arithmetic Logic Unit), etc. The controller generated different signals to the modules in the datapath based on different instructions.
Teaching Management System
  • A collaborative project.
  • Front-end: Simulated a real university educational management system using the AMIS architecture and connected to the back-end MySQL database through Flask.
  • Back-end: Built tables for students, courses, classes, teachers, etc., and designed an ER diagram. Implemented database functions based on requirements, such as teachers grading, student course enrollment, and more.
